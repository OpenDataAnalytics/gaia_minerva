Gaia-Minerva: A Girder plugin for running geospatial analyses in Minerva via the Gaia library
=====================================================

What are Gaia and Minerva?
----------------

`Minerva`_ is an open source platform for geospatial visualization using `Girder`_ as its datastore.

`Gaia`_ is a Python module for running geospatial analyses on vector and raster data.

Gaia-Minerva is a plugin for running Gaia analyses on vector data (primarily GeoJSON) in Minerva.


.. _Girder: https://girder.readthedocs.org
.. _Gaia: https://gaia.readthedocs.org
.. _Minerva: https://minerva.readthedocs.org
